## 流程
### 销售流程
- 现状（纸质）：
客户询价 -> 人工翻本查价/心算 -> 人工找货（凭记忆） -> 手写单据/计算器算总价 -> 收款/记账（欠条） -> 发货
- 风险：价格记错、找货慢、欠款漏记、库存不准。
- 目标（系统）：
客户询价 -> 系统搜索（支持模糊/规格）自动显示价格 -> 系统显示库位（A-03-2） -> 扫码/输入数量自动计算 -> 打印单据/微信发送 -> 自动扣减库存 & 记录应收 -> 发货
- 价值：秒级报价、精准定位、自动算账、实时库存。
### 进货流程
- 现状（纸质）：
厂家送货 -> 人工清点 -> 随意堆放 -> 在本子上记一笔“某月某日，某厂，某货，多少数量”
- 风险：不知道放哪了、不知道进价趋势、无法追溯批次。
- 目标（系统）：
厂家送货 -> 扫描/输入单据 -> 系统自动匹配历史进价（异常提醒） -> 指派/确认库位 -> 打印库位标签（可选） -> 确认入库（库存+，应付+）
- 价值：价格监控、库位绑定、供应商对账清晰。
