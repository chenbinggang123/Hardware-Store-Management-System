
# 五金店管理系统原型设计

## 1. 主界面说明
主界面底部有三个主要按钮，分别为：
- 首页
- 分析页
- 设置页

页面底部采用Tab栏布局，用户可随时切换到首页、分析页或设置页。

## 2. 页面跳转结构示意

```mermaid
graph TD
  登录/注册 --> 首页
  首页 --> 商品管理
  首页 --> 供应商管理
  首页 --> 客户管理
  首页 --> 进货管理
  首页 --> 销售管理
  首页 --> 库存管理
  分析页 --> 报表与统计
  设置页 --> 系统设置
```

## 3. 主界面结构示意

```mermaid
flowchart TB
  主界面 --> 首页
  主界面 --> 分析页
  主界面 --> 设置页
```
